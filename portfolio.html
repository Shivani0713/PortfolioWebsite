Preview : To Do App

The Notes App is a full-stack web application for creating and managing personal notes. Built using FastAPI and SQLite, it ensures fast performance and simple deployment. Users can sign up, log in securely using JWT-based authentication. Once logged in, they can create, edit, delete, and view notes. Each note can include a title, description, and an optional image. Images are uploaded using FastAPI's UploadFile and stored locally. The frontend is built with vanilla JavaScript, HTML, and CSS, offering a clean UI. JWT tokens are stored in localStorage to protect private routes. The app is lightweight, fast, and ideal for solo productivity use. It demonstrates practical integration of authentication, CRUD, and file handling.
